- Invariant:
  - Distance helpers return squared distance, not linear distance, unless the public wrapper explicitly takes a square root to subtract radius for capsule/sphere-style surface distance (`.ref/PhysX_4.0/physx/source/geomutils/src/distance/GuDistancePointSegment.h:68`, `.ref/PhysX_4.0/physx/source/geomutils/src/distance/GuDistancePointBox.cpp:46`, `.ref/PhysX_4.0/physx/source/geomutils/src/distance/GuDistancePointBox.cpp:65`, `.ref/PhysX_4.0/physx/source/geomutils/src/distance/GuDistancePointTriangle.h:118`, `.ref/PhysX_4.0/physx/source/geomutils/src/GuGeometryQuery.cpp:261`, `.ref/PhysX_4.0/physx/source/geomutils/src/GuGeometryQuery.cpp:272`).
